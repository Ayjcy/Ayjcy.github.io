docker

1.docker初始环境

虚拟机需要链接网络，安装yum工具

yum install -y yum-utils device-mapper-persistent-data lvm2 –skip-broken

2.配置网卡转发

（1）写入

cat < /etc/sysctl.d/docker.conf

net.bridge.bridge-nf-call-ip6tables = 1

net.bridge.bridge-nf-call-iptables = 1

net.ipv4.conf.default.rp_filter = 0

net.ipv4.conf.all.rp_filter = 0

net.ipv4.ip_forward=1

EOF

（2）重新加载内核参数

modprobe br_netfilter

sysctl -p /etc/sysctl.d/docker.conf

3.利用域名进行docker安装

（1用阿里云提供的docker专属repo仓库

curl -o /etc/yum.repos.d/docker-ce.repo http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

（2）更新yum缓存

yum clean all && yum makecache

（3）查看源中可用版本

yum list docker-ce –showduplicates | sort -r

（4）yum安装

（5）查看docker版本，验证是否成功

docker -v

（6）配置镜像加速器

mkdir -p /etc/docker

（7）启动docker

1）一定要关闭防火墙！！！

systemctl stop firewalld

2）docker配置文件重新加载

systemctl daemon-reload

3）设置开启自启动

systemctl enable docker

4）启动docker服务

systemctl start docker

（8）查看docker信息

\1) 查看 Docker 守护进程的详细信息

docker info

\2) 列出当前正在运行的容器

docker ps

3）列出当前系统中所有可用的 Docker 镜像

docker images

4）查找并返回指定的命令在系统中的安装路径

which docker

5）列出当前所有进程，并在输出中过滤包含关键字 docker 的进程信息

ps aux |grep docker

6）查找、检查 containerd 服务的运行状态

ps aux|grep containerd

systemctl status containerd

4.用docker运行nginx

（1）搜索一下远程仓库中的镜像文件是否存在

docker search nginx

（2）拉取，下载镜像

docker pull nginx

（3）查看镜像

docker images

（4）运行镜像

运行出具体内容，在容器中就跑着一个nginx服务

docker run -d -p 80:80 nginx

（5）查看容器是否在运行

docker ps

（6）访问网站

192.168.88.161:80

（7）停止容器

docker stop 17e4e6b1f7ab

（8）恢复容器

docker start 17e4e6b1f7ab

5.docker应用部署

（1）部署mysql

1）搜索mysql镜像

docker search mysql

2）拉取mysql镜像

docker pull mysql:5.6

3）创建容器，设置端口映射、目录映射

在/export/data/docker-data/目录下创建mysql目录用于存储mysql数据信息

mkdir -p /export/data/docker-data/mysql

cd /export/data/docker-data/mysql

docker run -id -p 3306:3306 –name=bigdata_mysql -v $PWD/conf:/etc/mysql/conf.d -v $PWD/logs:/logs -v $PWD/data:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=hadoop mysql:5.6

4）进入容器，操作mysql

docker exec –it bigdata_mysql /bin/bash

（2）部署Tomcat

1）搜索tomcat镜像

docker search tomcat

2）拉取tomcat镜像

docker pull tomcat

3）创建容器，设置端口映射、目录映射

创建目录tomcat，并进入目录下

docker run -id –name=c_tomcat -p 8080:8080 -v $PWD:/usr/local/tomcat/webapps tomcat

4）使用外部机器访问tomcat

（但是当前资源未公开，不过配置成了）

（3）部署Redis

1）搜索redis镜像

docker search redis

2）拉取redis镜像

docker pull redis:5.0

3）创建容器，设置端口映射

docker run -id –name=c_redis -p 6379:6379 redis:5.0
